Individoul couses

IT Academy STEP offers individual computer courses to learn or improve particular computer skills. Computer courses are conducted in an individual form, that is, one on one with a teacher. The first lesson is a trial: you can get acquainted with a teacher, ask questions of interest and discuss your wishes and available time. The average duration of courses is 2-3 months. At the end of the course, students receive IT Academy STEP certificates. In case if you take an authorized course – you also receive the corresponding international professional certificate.

PC user
This course is designed for students who want to learn how to use computers and does not have any experience. You will learn how to work with Windows operating system, use Microsoft Office programs like Word, Excel and Powerpoint, as well as working on the Internet and other software for confident everyday use of a PC. Students will also receive IT Academy STEP certificate upon completion of the course.

Course curriculum:

☑ Introduction to Windows
☑ Working with the keyboard. Navigating the file system
☑ Manipulating with files and foldersiOS;
☑ Work with the main menu of Windows OS. Work with utilities.
☑ Work in the local computer network
☑ Basic skills in the text editor MS Word. Adjust the page settings. Set and edit text
☑ Manipulation with text fragments. Using basic formatting tools.
☑ Professional documents formatting
☑ Working with tables and graphic elements
☑ Printing a document. Special features of MS Word.
☑ Introduction to PowerPoint
☑ Basic working skills with MS Excel
☑ Working with Formulas and Functions
☑ Registration of spreadsheets. Manipulation with cells
☑ Work with the Internet. Browsers and main settings and functions
☑ E-mail set up.

HTML 5.0 (XHTML)/CSS 3 (basic)
HTML – the language of hypertext markup which main task is to create a logical structure of a web page.
The HTML 5.0 / CSS 3 course introduces students to the latest version of the hypertext markup language and cascading style sheets (CSS) for web development.
Learning HTML and CSS is very important for everyone who is interested in learning web technologies.
During the course you will learn how to build a website from a graphic layout to a finished html-page, as well as how to publish sites on the Internet, work with search engines and web analytics.
